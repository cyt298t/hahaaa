# 综合服务器工具箱

一个面向 Debian 12 的轻量级服务器工具箱脚本，纯 Bash 编写，交互式数字菜单操作。目前内置两大模块：**定期 Ping 监控** 与 **定期测 IP 质量**，后续会持续扩展。

## ✨ 功能特性

### 1. 定期 Ping 监控
- 添加多个 IP / 域名目标，可为每个目标加备注
- 后台 systemd 服务定时 ping（默认每 60 秒一次），自动记录 `时间, 目标, 延迟, 丢包状态`
- 数据按天自动清理（默认保留 7 天，可调）
- 查看结果支持 **时间范围筛选**：最近 1 小时 / 最近 1 天 / 全部
- 直观展示：
  - **概况一览**：样本数、丢包率、延迟均/最小/最大值，附彩色趋势条
  - **分段概览**：按小时分段，带彩色圆点快速判断好坏，可钻取查看每条明细
- 颜色判定：🟢 <80ms　🟡 80–200ms　🔴 >200ms / 超时

### 2. 定期测 IP 质量
- 每天凌晨 05:00 自动执行 [IP.Check.Place](https://check.place) 检测并归档
- 原样保存完整报告（保留颜色），自动过滤进度动画等无用输出
- 检测结果默认保留 30 天，可调
- 查看结果：按时间列出所有报告，可单条查看或一次性按顺序展示全部
- **长期保留**：可将重要报告标记为 `[永久]`（移入 `keep/` 目录），永不被自动清理，也可随时取消
